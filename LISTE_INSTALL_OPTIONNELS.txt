NPOAP - Composants optionnels (apres installation.bat)

Executer les scripts depuis le dossier NPOAP (copie d'installation ou deploiement).

  install_msvc_build_tools.bat     Visual C++ Build Tools / MSVC (souvent requis pour PHOEBE2)
  install_cmake.bat                CMake (build C++/KBMOD)
  install_wsl.bat                  Sous-systeme Windows pour Linux
  install_ubuntu_wsl.bat         Distribution Ubuntu dans WSL
  install_astrometry_wsl.bat     Paquet astrometry.net dans WSL
  install_kbmod_wsl.bat          KBMOD sous WSL (installation auto via scripts/install_kbmod_wsl.sh)
  Installation_fsps/prospector.bat           Prospector + FSPS via WSL ^(manuel ; plus dans install_optionnels.bat^)
  INSTALLER_SORA_ASTROENV.bat                Depannage : mettre a jour sora-astro seul (sinon inclus dans requirements_install_optionnels.txt)
  requirements_install_optionnels.txt         PHOEBE, rebound, ultranest, SORA (sora-astro), SNe Ia, STDPipe+GP
                                             installe en une etape par install_optionnels.bat
                                             ou : pip install -r requirements_install_optionnels.txt
  requirements-cosmology-sne.txt             Compat : SNe Ia seul (anciennes copies du script)
                                             preferer requirements_install_optionnels.txt
